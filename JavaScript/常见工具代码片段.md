###  常见工具代码片段

- 判断浏览器类型

  ```js
  function myBrowser() {
      var e = navigator.userAgent;
      return e.indexOf("OPR") > -1 ? "Opera" : e.indexOf("Firefox") > -1 ? "FF" : e.indexOf("Trident") > -1 ? "IE" : e.indexOf("Edge") > -1 ? "Edge" : e.indexOf("Chrome") > -1 ? "Chrome" : e.indexOf("Safari") > -1 ? "Safari" : void 0
  }
  ```
