# String

### String.startsWith  和 String.endsWith

`startsWith` 和 ` endsWith` 都是对字符串的检索，返回一个Boolean；

`startsWith`  判断字符串是否以某个字符串开始；

```js
const str = 'gitbooks';
str.startsWith('git'); // true
```



`endsWith` 判断字符串是否已某个字符串结束；

```js
const str = 'gitbooks';
str.startsWith('books'); // true
```
